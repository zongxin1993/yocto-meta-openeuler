openEuler Embedded结合qemu使用
################################

使用qemu的仿真模式（tcg模式）可以快速的在自己的机器上启动一个Linux镜像。如果你想快速搭建一个openEuler Embedded环境，使用qemu可以屏蔽硬件和操作环境的差异，让你获得一个可运行的OS。
